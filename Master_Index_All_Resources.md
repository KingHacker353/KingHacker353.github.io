## â“ Common Issues

### "Tools nahi install ho rahe"
**Solution**: 
- Check internet connection
- Update system: `sudo apt update && sudo apt upgrade`
- Install dependencies: `sudo apt install golang-go python3-pip`
- Follow installation guide in main guidebook

### "Rate limiting aa rahi hai"
**Solution**:
- Use delays: `--delay 100ms` in gobuster
- Use proxy chains: `proxychains command`
- Reduce thread count: `-t 5` instead of default
- Reference: Card 29 in Quick Reference

### "False positives mil rahe hain"
**Solution**:
- Manual verification karo
- Multiple tools se confirm karo
- Screenshots lo proof ke liye
- Reference: Case studies for verification methods
